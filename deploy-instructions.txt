1. DEPLOY BACKEND (Fly.io):
   cd backend
   flyctl auth login
   flyctl secrets set JWT_SECRET="$(openssl rand -base64 32)" JWT_REFRESH_SECRET="$(openssl rand -base64 32)"
   flyctl launch --no-deploy
   flyctl deploy

2. DEPLOY FRONTEND (Vercel):
   - Go to https://vercel.com/dashboard
   - Click "Add New Project"
   - Import your GitHub repo
   - Add env var: VITE_API_BASE_URL=https://physician-dashboard-backend.fly.dev/api
   - Deploy!

3. UPDATE CORS:
   cd backend
   flyctl secrets set CORS_ORIGIN="https://your-app.vercel.app"
