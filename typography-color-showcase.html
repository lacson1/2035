Body Text Example

This is body text using the Inter font family. The default size is 1rem (16px) with a line height of 1.5rem (24px) and letter spacing of -0.01em for optimal readability.

The consistent typography system ensures that all text elements across the application maintain visual harmony. Headings use progressively larger sizes with tighter letter spacing, while body text maintains comfortable reading spacing.

All text elements inherit the Inter font family, which provides excellent readability at all sizes and weights. The font is loaded from Google Fonts with weights 300 through 800, ensuring consistent rendering across all devices.
