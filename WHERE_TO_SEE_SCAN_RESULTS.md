## ğŸ’¡ Tips

### Finding Specific Results:
1. **Use Search**: Type patient name, study type, or body part
2. **Filter by Modality**: Show only CT, MRI, X-Ray, etc.
3. **Sort by Date**: Most recent results appear first
4. **Check Status**: Make sure status is "Completed"

### Understanding Results:
- **Findings Text**: Shows radiology interpretation
- **Status**: Must be "Completed" to view images
- **Report URL**: If provided, links to external DICOM viewer
- **Date**: When study was performed (not when ordered)

### If Results Don't Appear:
1. Check study status is "Completed"
2. Verify patient is selected
3. Check filters aren't hiding results
4. Refresh the page
5. Check backend connection
